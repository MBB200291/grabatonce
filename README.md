# Grab at Once!

Chrome extension that allow downloading files in one go.

## Usage

- 建議在[設定]("chrome://settings/downloads")頁面中，關閉"下載每個檔案前先詢問儲存位置"功能。
  ![1](chrome-setting-download.png)
